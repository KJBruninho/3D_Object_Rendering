# 3D Object Rendering 🧊

Um motor de renderização 3D simplificado desenvolvido em **C++**, focado no processamento e visualização de geometrias tridimensionais.

## Sobre o Projeto
Este projeto explora os fundamentos da computação gráfica, implementando algoritmos para ler, processar e renderizar objetos 3D. O foco principal é a compreensão do pipeline de renderização, transformações lineares (translação, rotação, escala) e a projeção de coordenadas 3D para um espaço 2D.

## Tecnologias Utilizadas
* **Linguagem:** C++
* **Ambiente de Desenvolvimento:** Visual Studio (ficheiro `.sln` incluído)
* **Conceitos:** Álgebra Linear, Computação Gráfica, Pipeline de Renderização.

## Estrutura do Repositório
* `/Project_2`: Contém o código-fonte principal e os ficheiros de lógica de renderização.
* `Project_2.sln`: Solução do Visual Studio para gestão do projeto.

## Como Executar
1.  Clona o repositório:
    ```bash
    git clone [https://github.com/KJBruninho/3D_Object_Rendering.git](https://github.com/KJBruninho/3D_Object_Rendering.git)
    ```
2.  Abre o ficheiro `Project_2.sln` no **Visual Studio**.
3.  Certifica-te de que as dependências de sistema (caso existam) estão configuradas corretamente.
4.  Compila e executa o projeto.
